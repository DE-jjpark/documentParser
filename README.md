# documentParser

A document parsing toolkit.

## Requirements

- Python 3.12
- [uv](https://docs.astral.sh/uv/)

## Setup

```bash
uv sync
```

## Usage

```bash
uv run document-parser --help
```

## Development

```bash
uv run pytest        # run tests
uv run ruff check    # lint
uv run ruff format   # format
```
